CSC110 Syllabus

Course Material

Most material will be covered by in-class lectures. I may occasionally post reading assignments or videos to supplement what we cover in class.

Secondary Resources

A few free resources you may want to use for further reading or extra practice:

  • MOOC.fi: The University of Helsinki allows free access to material from many of their courses, including an introductory Java class. This course provides reading material and practice exercises that can be accessed after you make an account.
  • UMN Textbook: The University of Minnesota's Open Textbook Library has free textbooks, including Introduction to Programming Using Java.
  • LibreText: The website libretexts.org has textbooks that cover a lot of the material we'll be learning this semester. Two that I've used in the past are:
    • Programming and Computation Fundamentals, by Busbee and Braunschweig
    • Computer Science from the Bottom Up, by Wienand

Software

There are several programs you will need for this class: a bash/zsh terminal, a text editor, and the Java 25 JDK. There are guides linked on Blackboard for installing these programs, and we will cover them during the first week of class.

Other Resources

The Java API is a useful reference. It may be intimidating right now, but the more you learn about Java and programming in general, the more easily you'll be able to navigate the API and use it to look up information.

If you're curious about how Java works, you may want to look at the Java Specification. It's a very technical document and not likely to come up while you're coding, but I might reference it in class and some of you may find it interesting.

Finally, asking good questions is one of the most useful skills you can have as a student. In my experience, most people (myself included) don't ask questions as often as they should, and it can be difficult to ask a question that will quickly lead to a helpful answer. I think the blog posts below provide excellent advice for asking and answering questions:

  • https://jvns.ca/blog/good-questions/
  • https://jvns.ca/blog/2021/10/21/how-to-get-useful-answers-to-your-questions/
  • https://jvns.ca/blog/answer-questions-well/
