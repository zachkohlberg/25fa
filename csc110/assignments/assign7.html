Score Categories

The game allows the user to choose from ten different score categories:

  1. Aces
  2. Twos
  3. Threes
  4. Fours
  5. Fives
  6. Sixes
  7. 3 of a Kind
  8. 4 of a Kind
  9. Yahtzee
  10. Chance

The rules for calculating these scores are as follows:

  • The first six categories (the "upper categories" in Yahtzee) are scored the same as in previous assignments: the category number multiplied by how many dice equal that number.
  • The 3 and 4 of a kind categories follow similar rules:
    • If the user has at least three (3 of a kind) or at least four (4 of a kind) dice with the same number, then these categories are worth the sum of all dice (not just the matching dice).
    • If the user doesn't meet the 3/4 of a kind requirement, then these categories are worth 0 points.
  • The Yahtzee category is worth 50 points if all of the dice are the same and 0 points otherwise.
  • The chance category is always worth the sum of all dice.

I/O

The expected format for printing scores is:

Score Categories:

1. Aces: X
2. Twos: X
3. Threes: X
4. Fours: X
5. Fives: X
6. Sixes: X
7. 3 of a Kind: X
8. 4 of a Kind: X
9. Yahtzee: X
10. Chance: X

Note the blank line between "Score Categories:" and the list. Each "X" should be replaced with the score the user would receive for that category. You can print this with a loop if you put the category names in an array and iterate over the array, but remember that the numbers next to the categories are 1 greater than the corresponding array index. Here's some code to create an array with all the category names:

String[] names = new String[] {

    "Twos",
    "Threes",
    "Fours",
    "Fives",
    "Sixes",
    "3 of a Kind",
    "4 of a Kind",
    "Yahtzee",
    "Chance"
};

Functions

  • main
    • parameters: none
    • return type: void
    • This function must print "Dice Game" as its only output and not attempt to read any input.
  • calculateScore
    • parameters: an int array representing five dice and an int representing the category (1 to 10)
    • return type: int
    • This function must return the score for the given category.
  • sumDice
    • parameters: an int array representing five dice
    • return type: int
    • This function must return the sum of all numbers in the array.
    • This is a helper function that you should call when scoring 3/4 of a Kind and Chance.
  • countMostCommon
    • parameters: an int array representing five dice
    • return type: int
    • This function must find the most common number in the array and return how many times that number appears in the array.
    • This is a helper function that you should call when scoring 3/4 of a Kind and Yahtzee to check whether the dice meet the requirement for that category.
    • Hint: you can use countDice to find out how many times each possible die value (1 to 6) appears in the array and return the largest count. Math.max or an if statement can take the larger of two numbers.
  • scoreUpper
    • parameters: an int array representing five dice and an int representing a die value
    • return type: int
    • Other than the name, this function should be identical to your calculateScore function from the previous assignment.
  • scoreChance, score3OfAKind, score4OfAKind, and scoreYahtzee
    • parameters: an int array representing five dice
    • return type: int
    • Each of these functions must calculate the score for their category according to the Score Categories requirements.
  • printScores
    • parameters: an int array representing five dice
    • return type: void
    • This function must print the score categories as specified in the I/O requirements.
  • playRound
    • parameters: none
    • return type: void
    • This should contain all the code from the previous assignment's main function. It will be given the exact same I/O tests used for main in the previous assignment.
