Assignment 5

In this assignment you will write some functions to help with coding your dice game. You won't really have a program at the end of this assignment, but you will get to use all of this code unmodified in the next assignment.

Autograded

This is an autograded assignment. Read the Autograded Assignments page before starting.

This program involves random numbers. Make sure you're following the random number generation requirements for autograded assignments so that your code is compatible with the tests.

Directions

  1. Write this program in a file named DiceGame1.java.
  2. Implement each of the required functions described in the requirements section. These functions will be used in the next assignment to make the first version of the dice game. For now, your code will just be a collection of functions rather than an interactive program.
  3. Test your functions by calling them from main and printing any relevant information. Before submitting, delete all code from main and just print "Dice Game". There's one test case for main, and it just expects to see this message.

Requirements

Most requirements are covered by the function descriptions below. The expected format for printing dice is:

 A   B   C   D   E
+-+ +-+ +-+ +-+ +-+
|1| |5| |3| |3| |6|
+-+ +-+ +-+ +-+ +-+

There must be no trailing whitespace on these lines, and it must end with a line break (println, not print). If you use a loop to print the die values, then you can include an if statement in the loop to control whether a space is printed after the last part:

loop (...) {
    // print the die and vertical lines
    IO.print(...);
    // only print a space if we're not on the last index of the array
    if (index < array.length - 1) {
        IO.print(" ");
    }
}
// end the middle line
IO.println();

Functions

  • main
    • parameters: none
    • return type: void
    • This function must print "Dice Game" as its only output and not attempt to read any input.
  • countDice
    • parameters: an int array representing five dice and an int representing the die value to count
    • return type: int
    • This function must return how many times the die value appears in the array.
  • printDice
    • parameters: an int array representing five dice
    • return type: void
    • This function must print the dice in the format specified earlier.
  • calculateScore
    • parameters: an int array representing five dice and an int representing the die value to score
    • return type: int
    • This function must return the score for the given die value, which is equal to the die value multiplied by the number of times it appears in the array.
    • Hint: call your countDice function to figure out how many times the die value appears in the array, then multiply this count by the die value.
  • rollAllDice
    • parameters: an int array representing five dice
    • return type: void
    • This function must replace each value in the array with a random number from 1 to 6.
    • The random numbers must be generated in the order they will appear in the array to ensure compatibility with the tests.
  • rollSomeDice
    • parameters: an int array representing five dice and a boolean array representing whether the die at the same index should be rolled
    • return type: void
    • This function must replace each value in the int array with a random number from 1 to 6 if and only if the element at the same index in the boolean array is true.
    • The random numbers must be generated in the order they will appear in the array to ensure compatibility with the tests.
  • parseReroll
    • parameters: a String
      • you may assume this string contains zero or more of the lowercase letters from 'a' to 'e'
      • the letters may be in any order, so don't assume the first character is 'a'
    • return type: a boolean array
    • This function must return an array of five booleans, each indicating whether a letter appeared in the string. Each index of the array corresponds to a different lowercase letter:
      • at index 0: whether 'a' appears in the string
      • at index 1: whether 'b' appears in the string
      • at index 2: whether 'c' appears in the string
      • at index 3: whether 'd' appears in the string
      • at index 4: whether 'e' appears in the string
    • Hint: you can check whether a character appears in a string with the string's indexOf method. If the index is -1, then the character is not in the string. If the index is anything else, then the character is in the string.
