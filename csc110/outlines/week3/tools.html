CSC110 Lecture Outlines

Tools

This is a summary of all the tools currently at your disposal. Right now, these are all valid inside of the main function of a compact Java program. These examples include many placeholders, which I've surrounded with underscores and written in lower_snake_case. For example, _type_ is a placeholder that can be any of the types we've learned about: int, double, or String. _function_identifier_ is a placeholder than can be replaced with the identifier (name) of any function we've learned about, such as println or abs.

// Types (_type_): int, double, String
// Operators (_op_): +, -, *, /, %
// Identifiers (_identifier_): starts with non-digit, only contains letters,
//                             digits, $, or _
// Namespaces (_namespace_): IO, Integer, Double, Math

////////////////
// Statements //
////////////////

// Variable declaration
_type_ _identifier_;
// Example:
int age;

// Variable declaration and initialization
_type_ _identifier_ = _expression_;
// Example:
String name = "My Name";

// Variable declaration (infers type) and initialization
var _identifier_ = _expression_;
// Example:
var name = "My Name";

// Assignment
_identifier_ = _expression_;
// Example:
age = age + 1;

// Function call (usually a void function)
_namespace_._function_identifier_(_arguments_);
// Example:
IO.println("Hello World");

/////////////////
// Expressions //
/////////////////

// int literal examples:
20
-3
1_000_000

// double literal examples:
20.0
-3.14
1_000_000.0

// String literal examples:
"Hello World"
"abc"
""

// Variable
_variable_identifier_
// Example:
age

// Function call (non-void function)
_namespace_._function_identifier_(_arguments_)
// Example:
Integer.parseInt("100")

// String method call
_string_expression_._method_identifier_(_arguments_)
// Example:
"Hello World".toUpperCase()

// int cast
(int)_double_expression_
// Example:
(int) (1000.0 / 7.0)

// Parenthesis (for order of operations)
(_expression_)
// Example:
(number + 10)

// Binary operator
_expression_ _op_ _expression_
// Example:
"Hello " + "World!"

////////////////////////
// Specific Functions //
////////////////////////

// Return type: void
IO.print(_expression_)
IO.println(_expression_)
IO.println()

// Return type: String
IO.readln(_string_expression_)
IO.readln()

// Return type: int
Integer.parseInt(_string_expression_)
Math.abs(_int_expression_)
Math.min(_int_expression_, _int_expression_)
Math.max(_int_expression_, _int_expression_)

// Return type: double
Double.parseDouble(_string_expression_)
Math.abs(_double_expression_)
Math.min(_double_expression_, _double_expression_)
Math.max(_double_expression_, _double_expression_)
Math.pow(_double_expression_, _double_expression_)

/////////////////////////////
// Specific String Methods //
/////////////////////////////

// Return type: String
_string_expression_.toLowerCase()
_string_expression_.toUpperCase()
_string_expression_.substring(_int_expression_)
_string_expression_.substring(_int_expression_, _int_expression_)

// Return type: int
_string_expression_.length()
