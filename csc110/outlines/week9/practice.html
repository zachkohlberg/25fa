CSC110 Lecture Outlines

Practice

  1. Review the example programs and answer any questions in the comments.
  2. Write a function to parse days of the week into the corresponding number from 1 to 7, with weeks beginning on Sunday. Allow three-letter abbreviations for each day and make the function case-insensitive.
  3. Read a string from the user and print a new string where every character that isn't a letter or digit is replaced with an underscore. Don't use any built-in methods or functions other than charAt.
  4. Prompt the user for input in the format "amount unit1 unit2", where amount is a double and unit1 and unit2 are either "mm" (millimeter), "cm" (centimeter), "m" (meter), or "km" (kilometer). Each part of the input is separated by exactly one space. Convert the amount from unit1 to unit2, printing the result rounded to two decimal places followed by the unit. For example, given the input "307.8 cm m" your program would print "3.08 m".
  5. Write a function that takes a tic-tac-toe board array (a 3x3 int[][] containing 0 for empty spaces, 1 for Xs, and 2 for Os) and return whether the X player has won. You can break this into separate, smaller questions and check each of them separately:
    • Does any row have three Xs?
    • Does any column have three Xs?
    • Does either diagonal have three Xs?
