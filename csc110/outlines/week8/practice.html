CSC110 Lecture Outlines

Practice

  1. Answer the questions in the comments of the example programs.
  2. Given two integers a and b, write a boolean expression that is true when a / b is greater than b / a and false otherwise. If either number is zero, then the condition must evaluate to false without triggering an exception. Hint: take advantage of short-circuiting!
  3. Write a function that:
    • takes two int parameters, a lower bound and an upper bound
    • returns a random int between the lower bound (inclusive) and the upper bound (exclusive)
    • generates its random value with Math.random() and adjusts it to be in the correct range with an equal chance of being any number in that range
  4. Write a function that compares two int arrays and returns whether they are equal (same size, same elements in the same order).
  5. Write two versions of an "increment array" function:
    • incrementArrayCopy takes an int[] parameter and returns a new int[] of the same length where each element is equal to the corresponding element in the parameter plus one
    • incrementArrayMutate takes an int[] parameter and returns nothing (void), but it increases the value of each element in the parameter by one
