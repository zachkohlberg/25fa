Classes

  • Java defines objects with classes
  • some OOP languages use prototypes instead of classes (Javascript)
  • class keyword in Java has multiple uses:
    • define a new type of object
    • create a namespace to organize static functions
    • can be used for either or both at the same time
  • defining a type
    • instance variables: each instance has its own copy of these variables
    • constructor: special type of function, initializes instance variables
    • methods: built-in functions we can call on instances of the class
  • instantiating/using our type
    • instantiate: create an instance of a class, new followed by calling constructor
    • variables/methods: access class members with "dot syntax"
// PointDemo.java
void main() {
    Point a = new Point(5, 10);
    Point b = new Point(0, 0);

    double distance = a.distanceTo(b);
    a.printMe("a");
    b.printMe("b");
    IO.println("Distance from a to b: " + distance + ".");
}

// declare class
class Point {
    // declare instance variables
    double x;
    double y;

    // constructor
    Point(double initX, double initY) {
        // initialize instance variables
        this.x = initX;
        this.y = initY;
    }

    // method to calculate distance to a second point
    double distanceTo(Point other) {
        double dx = this.x - other.x;
        double dy = this.y - other.y;
        return Math.sqrt(dx * dx + dy * dy);
    }

    void printMe(String label) {
        IO.println(String.format("%s(%.3f, %.3f)", label, this.x, this.y));
    }
}
