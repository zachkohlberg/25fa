System

  • System class contains commonly-used functions and standard i/o streams
  • before Java 25
    • use the System.out and System.err output streams to print to standard out and standard err
    • create a Scanner or other reader object to read standard in from System.in
  • after Java 25
    • added the IO class with convenient static methods for printing to standard out and reading from standard in
  • other useful features of System:
    • System.exit(code): exit the running Java program, with a zero code indicating your program succeeded and a nonzero code indicating an error/failure
    • System.nanoTime(): get time in nanoseconds
      • not useful for checking current time, just how much time has elapsed since the last time you called System.nanoTime()
    • System.currentTimeMillis(): get milliseconds since the "Unix epoch" (one way to check the current time)
