Standard vs Compact Java

In this class, most (possibly all) of our programs will be written as Compact Source Files. It's not necessary to understand what exactly this means right now, but you can read the relevant JDK Enhancement Proposal if you want the details. For now, I'll summarize the main points:

  • Java normally requires a lot of boilerplate to write even the simplest program. That boilerplate code involves Java features that a new programmer won't be equipped to understand.
  • Basic input and output (I/O) operations, which most of our programs will need, also require features that are difficult for new programmers to understand.
  • A compact source file is a more concise way of writing a Java program. It removes most of the boilerplate from small programs and provides easier access to I/O functions. This lets us focus on the actual program we're trying to write and introduce new concepts when we're ready for them.

This guide will cover how to convert standard Java programs into compact Java programs. This may be useful if:

  • You have prior Java experience and want a quick guide for converting the way you normally code to the way we code in this class.
  • You're looking at an example that doesn't use the new compact source features. These features are new, so most textbooks and online examples don't use them.

1. Remove the Class

Standard Java programs will have most of their code inside of a class:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}

We don't need the class:

public static void main(String[] args) {
    System.out.println("Hello World");
}

2. Simplify Main

The main function is more complex in a standard Java program:

public static void main(String[] args) {
    System.out.println("Hello World");
}

All we need is void main():

void main() {
    System.out.println("Hello World");
}

3. Output

Standard Java programs use System.out. or System.err. followed by println, print, or printf for output:

System.out.println("Regular output.");
System.err.println("Error message.");

We use IO.println and IO.print for output:

IO.println("Regular output.");
IO.println("Error message.");

To finish converting the earlier Hello World example:

void main() {
    IO.println("Hello World");
}

4. Input

Standard Java programs need to create a special object (such as a scanner or buffered reader) to read user input:

java.util.Scanner stdin = new java.util.Scanner(System.in);

System.out.print("Please enter some input: ");
var input = stdin.nextLine();

System.out.println("Please enter more input: ");
var input2 = stdin.nextLine();

We use IO.readln with an optional prompt message:

var input = IO.readln("Please enter some input: ");
var input2 = IO.readln("Please enter more input: ");
